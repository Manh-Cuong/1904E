sự khác biệt của include, include_once, require, require_once

    sự giống nhau: 4 hàm này đều dùng để nạp 1 file vào 1 file khác
    sự khác nhau:
    -include: khi mà quá trình nạp file bị lỗi thì chương trình vẫn chạy tiếp
    -include_once: giống include nhưng chỉ nạp file 1 lần ngay cả khi gọi nhiều lần
    -require: khi mà quá trình nạp file bị lỗi thì chương trình sẽ dừng chạy
    -require_once: giống require nhưng chỉ nạp file 1 lần ngay cả khi gọi nhiều lần
