thuộc tính id trong html 

    -id trong html có giống như CMND của 1 người
    nó sẽ duy nhất trong html và k được phép trùng giữa các thẻ
    id viết liền k dấu, k có kí tự đặc biệt
    -class để gom nhóm giữa các thẻ html có điểm chung
    1 thẻ html có thể có 1 hoặc nhiều class khác nhau
    các class được phân tách bởi dấu cách
    quy tắc tên class viết liền k dấu k có kí tự đặc biệt
    -mục đích của thuộc tính class và id là để kết hợp với các ngôn ngữ như css, js để thực hiện các tác vụ trên trang web
