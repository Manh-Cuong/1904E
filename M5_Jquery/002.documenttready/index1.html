jquery document ready
    selector là bộ chọn trong jquery để chọn được thẻ html dựa trên class id tag name
    selector bắt đầu bằng:$("Selector") hoặc $('selector')
    riêng document hoặc window thì k cần đặt trong nháy kép
    cú pháp document ready $(document).ready();
    bên trong ready sẽ là 1 callback js
    callback js chính là 1 hàm nhưng ko có tên function(){//code}
    Callback có nghĩa là hành động sẽ thực hiện sau 1 hành đông nào đó
    Ví dụ khi mà document bao gồm các thẻ html , css , js , hình ảnh được load xong
    thì callback trong $(document).ready(); mới chạy
